# notes-zho-bet365

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涵盖日常笔记、信息整理、静态展示等用途，不针对任何特定域名或网站，属于通用型静态文件仓库。

## 目录结构

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源文件（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放各个独立的 HTML 文件，每个文件为一个完整页面。
- `assets/` 目录用于存放各页面可能共享的静态资源。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开查看。
- 页面之间无强依赖关系，可单独使用或通过索引页面导航。
- 归档内容以信息整理与静态展示为主，不涉及动态交互或后端服务。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过 commit 记录体现。
- 如发现页面内容错误、链接失效或格式问题，欢迎提交 Issue 或 Pull Request。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 如需部署到静态托管服务，可将仓库根目录作为站点根目录。
